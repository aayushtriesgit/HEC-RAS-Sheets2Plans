# HEC-RAS-Sheets2Plans
A Python desktop app for managing HEC-RAS project plans. It provides a spreadsheet interface to view, edit, and generate .pXX plan files, including geometry and flow assignments. It automates plan creation and updates .prj files for faster, structured hydraulic modeling workflows.

HydroRAS Toolkit is a Python-based desktop application for managing HEC-RAS project files through an interactive spreadsheet interface.

## Features
- View and edit HEC-RAS plans
- Assign geometry and flow files using dropdowns
- Auto-generate new plan files
- Update `.prj` file automatically
- Undo system with full rollback support

## Usage
1. Run `HydroRAS_Toolkit.exe`
2. Select HEC-RAS project folder
3. Edit plans in spreadsheet
4. Click RUN to generate/update plans
5. Use Undo if needed

## Requirements (for source)
- Python 3.10+
- tksheet
- tkinter
